## 项目简介

模仿今日头条的财经类新闻微信小程序，可以二次开发，可以商用。
这个项目是前后端分离的java端，只提供api，如果想要完整运行项目还需要启动微信端。
里面内置的ruoyi-ui 是基于vue2的，之后可以直接删除。

## 内置功能

1. 首页推荐：根据热度推荐新闻
2. 用户收藏：收藏新闻
3. 用户点赞：点赞新闻
4. 用户评论：评论新闻内容
5. 分类咨询：给新闻进行了分类整理
